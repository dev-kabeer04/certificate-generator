Course Results (Percentages)

These results are used to automatically select certificate templates based on the learner's percentage.

Upload CSV

Required columns: name or email (at least one). Optional: percentage, completed.

Example CSV:
name,email,percentage,completed
John Doe,john@example.com,78.5,1
Jane Smith,jane@example.com,92,yes

Manual Entry

Example lines:
John Doe,john@example.com,78.5,1
Jane Smith,jane@example.com,92,yes
Ali Khan,ali@example.com

How this is used: When generating certificates for a course the system looks up the latest result for each recipient (matching by email first, then name) and maps percentage to a template. If no result is found the Generate page template is used as fallback.
